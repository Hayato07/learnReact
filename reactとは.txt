#01 Reactとは何か

from 公式

## Reactは、ユーザーインターフェイスを構築するためのライブラリ

Reactの特徴
- Declarative
対話的なUIが比較的作りやすい
アプリケーションが持つそれぞれの状態がシンプルで分かりやすい
データ更新があった場合、効率的に狙ったコンポーネントを更新&レンダリングを行う

- Component-Based
それぞれの状態を管理するカプセル化されたコンポーネントを組み合わせて複雑なUIを実現する
コンポーネントのロジックはJavaScriptで書かれているので、アプリケーションのデータをDOMと関連づけずに(DOMのことは考えずに)状態管理ができる

- Learn Once, Write Anywhere
React NativeやReact VRなどがあるよ、という意味？

## 簡単なコンポーネント 

Reactコンポーネントは、render()メソッドを実行します。renderメソッドは、入力dataを引数として受け取り、表示するものをreturnします。

JSXはオプションであって、必須ではありません。JSXを利用しない書き方でもReactを利用できます。
